Java教程第一章：写一个HelloWorld

对于如何配置开发环境，您大可在网上找到一些教程，这里我们将不再多说。在本章中，我们将使用记事本作为编辑器。

首先新建一个文件，取名为HelloWorld。注意后缀名是java。

我们在文件中输入以下代码：

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("HelloWorld");
  }
}

此时你可能会觉得自己被逼着输入了一些东西。不要慌，这些我们会在待会说。

public class HelloWorld用于声明一个类。“HelloWorld”应当为文件名。

public static void main(String[] args)是主方法。所有主类都会从这个方法开始执行。

System.out.println是打印语句。它可以打印出“HelloWorld”之类的玩意。
PS:你可能还注意到了System.out下还有一个print方法，具体的区别可以自己去试一下。

代码写好了，在终端中输入javac HelloWorld.java编译文件。如果什么都没有出现，说明编译成功！
