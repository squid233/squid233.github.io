上一节中我们成功配置了 JDK。本节简单介绍一下 JShell。

JShell 在 JDK 9 中加入，支持运行 Java 代码片段而无需 Java 源文件。

输入`jshell`即可启动，或者直接打开`bin`目录下的`jshell`也行。
